# Chrome Stock Monitor

**[English](./README_EN.md)**

轻量级 Chrome 股票监控扩展，支持 A 股、港股、美股实时行情和自定义提醒。

无需注册，无需后端，装上就能用。

## 功能

- **三市场行情**：A 股、港股、美股分开管理，一键切换
- **实时数据**：最新价格、涨跌额、涨跌幅，红涨绿跌
- **灵活搜索**：支持代码、名称、拼音搜索，A 股和港股走全市场检索
- **自动刷新**：后台持续运行，刷新间隔 1–30 秒可调
- **涨跌幅提醒**：设置阈值，涨跌幅达标时角标自动提示
- **目标价提醒**：设定价格方向（≥ / ≤），到了就提醒
- **Badge 提醒**：触发的股票数量直接显示在扩展图标上
- **拖拽排序**：长按股票卡片左侧 grip 手柄，拖动调整排列顺序

## 安装

```bash
git clone https://github.com/makersy/chrome-stock-monitor.git
```

1. 打开 Chrome，地址栏输入 `chrome://extensions/`
2. 右上角开启「开发者模式」
3. 点击「加载已解压的扩展程序」，选择项目文件夹
4. 固定插件到工具栏

## 使用

1. 点击扩展图标打开弹窗
2. 通过顶部 tab 切换市场（A 股 / 港股 / 美股）
3. 在搜索框输入股票代码或名称，点击添加
4. 点击 🔔 图标设置涨跌幅阈值或目标价
5. 点击 ⚙ 进入设置，调整自动刷新和提醒开关

## 数据来源

行情数据来自腾讯行情接口和东方财富搜索接口，均为公开免费接口，插件不收集或上传任何用户数据。
